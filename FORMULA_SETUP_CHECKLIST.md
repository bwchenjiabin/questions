## 🧪 功能测试

### 测试步骤

1. **启动开发服务器**
   ```bash
   npm run dev
   ```

2. **打开题目录入页面**
   - 导航到题目录入功能
   - 找到任意富文本编辑器（题干、选项等）

3. **测试插入公式**
   - 点击输入框进入编辑模式
   - 在工具栏找到 **"插入公式"** 按钮（fx 图标）
   - 点击按钮，应该弹出公式编辑对话框
   - 输入测试公式：`\frac{1}{2}`
   - 点击确定
   - 公式应该正确显示为 ½

4. **测试编辑公式**
   - 选中已插入的公式
   - 点击工具栏的 **"编辑公式"** 按钮
   - 修改公式为：`\frac{3}{4}`
   - 点击确定
   - 公式应该更新为 ¾

5. **测试保存和加载**
   - 插入几个公式后点击"确定"保存
   - 重新点击输入框
   - 公式应该正确显示

## ⚠️ 常见问题排查

### 问题 1：工具栏没有公式按钮
**原因：** 插件未正确注册
**解决：** 检查 `Boot.registerModule(formulaModule)` 是否执行

### 问题 2：点击公式按钮没反应
**原因：** KaTeX 未正确加载
**解决：** 检查 `import 'katex/dist/katex.min.css'` 是否存在

### 问题 3：公式显示为代码
**原因：** KaTeX 渲染失败
**解决：** 
1. 检查 katex 版本是否为 0.15.6
2. 清除缓存重新启动：`npm run dev`

### 问题 4：控制台报错
**检查项：**
1. 浏览器控制台是否有 CSS 加载错误
2. 是否有 JavaScript 错误
3. 网络请求是否正常
