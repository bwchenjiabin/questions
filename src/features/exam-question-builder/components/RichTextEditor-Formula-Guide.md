# 富文本编辑器 - 数学公式使用指南

## 功能说明

富文本编辑器现已集成数学公式插件，支持使用 LaTeX 语法输入数学公式。

## 关于公式输入框

点击"插入公式"按钮后，会弹出一个简洁的输入框，这是正常的设计。你需要在输入框中输入 LaTeX 代码，然后点击"确定"，公式就会被渲染成漂亮的数学符号。

**为什么是输入框而不是可视化编辑器？**
- LaTeX 是数学公式的标准语法，简洁高效
- 输入框设计轻量，不影响编辑器性能
- 配合公式辅助面板，可以快速插入常用符号

## 使用方法

### 方式 1：直接输入 LaTeX 代码

1. 点击工具栏中的 **"插入公式"** 按钮（fx 图标）
2. 在弹出的输入框中输入 LaTeX 公式
3. 点击确定，公式将插入到编辑器中

### 方式 2：使用公式辅助面板（推荐）

我们提供了 `FormulaHelper` 组件，包含常用符号和公式模板：

1. 在页面中添加 `<FormulaHelper>` 组件
2. 点击需要的符号，LaTeX 代码会自动复制
3. 在编辑器中点击"插入公式"，粘贴代码即可

### 编辑已插入的公式

1. 选中已插入的公式（或双击公式）
2. 点击工具栏中的 **"编辑公式"** 按钮
3. 修改 LaTeX 公式
4. 点击确定保存修改

## 常用 LaTeX 公式示例

### 基础运算符

| 符号 | LaTeX 代码 | 说明 |
|------|-----------|------|
| + | `+` | 加号 |
| - | `-` | 减号 |
| × | `\times` | 乘号 |
| ÷ | `\div` | 除号 |
| ± | `\pm` | 正负号 |
| = | `=` | 等号 |
| ≠ | `\neq` | 不等号 |
| < | `<` | 小于 |
| > | `>` | 大于 |
| ≤ | `\leq` | 小于等于 |
| ≥ | `\geq` | 大于等于 |

### 分数

```latex
\frac{1}{2}        # 二分之一
\frac{a+b}{c}      # (a+b)/c
```

### 根式

```latex
\sqrt{2}           # 根号2
\sqrt[3]{8}        # 三次根号8
```

### 上标和下标

```latex
x^2                # x的平方
x^{10}             # x的10次方
x_1                # x下标1
x_{i+1}            # x下标i+1
```

### 小学数学常用公式示例

#### 1. 四则运算
```latex
3 + 5 = 8
10 - 4 = 6
6 \times 7 = 42
20 \div 4 = 5
```

#### 2. 分数运算
```latex
\frac{1}{2} + \frac{1}{3} = \frac{5}{6}
\frac{3}{4} \times \frac{2}{5} = \frac{3}{10}
```

#### 3. 小数
```latex
3.14
0.5 + 0.3 = 0.8
```

#### 4. 百分数
```latex
50\%
\frac{1}{4} = 25\%
```

#### 5. 面积公式
```latex
S = a \times b              # 长方形面积
S = a^2                     # 正方形面积
S = \frac{1}{2} \times b \times h    # 三角形面积
```

#### 6. 周长公式
```latex
C = 2 \times (a + b)        # 长方形周长
C = 4a                      # 正方形周长
C = 2\pi r                  # 圆的周长
```

#### 7. 几何符号
```latex
\angle ABC                  # 角ABC
\triangle ABC               # 三角形ABC
\perp                       # 垂直
\parallel                   # 平行
```

#### 8. 单位换算
```latex
1km = 1000m
1m = 100cm
1kg = 1000g
```

## 高级用法

### 多行公式
```latex
\begin{aligned}
x + y &= 10 \\
x - y &= 2
\end{aligned}
```

### 矩阵
```latex
\begin{pmatrix}
1 & 2 \\
3 & 4
\end{pmatrix}
```

### 求和符号
```latex
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
```

## 注意事项

1. **LaTeX 语法**：公式使用标准 LaTeX 数学语法
2. **预览**：输入公式后会实时预览渲染效果
3. **编辑**：双击公式或选中后点击"编辑公式"按钮可重新编辑
4. **兼容性**：公式会被保存为 HTML 格式，可以正常显示和打印
